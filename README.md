# analiza_ecommerce

Kompleksowa Analiza Rocznych Danych Sprzedażowych

Do projektu wykorzystano dane sprzedażowe udostępnione przez użytkownika 'KeithGalli' (miesięczne pliki CSV dotyczące sprzedaży elektroniki w USA).
Oryginalne źródło danych: 'github.com/KeithGalli/Pandas-Data-Science-Tasks'

Technologia:
1. Język: Python
2. Biblioteki: Pandas, NumPy, Matplotlib 
3. Środowisko: Jupyter Notebook

Projekt miał na celu optymalizację strategii sprzedażowej i marketingowej na kolejny rok, poprzez analizę historycznych danych z 12 miesięcy.

Odpowiedziano na następujące pytania biznesowe:
1. Jaki był najlepszy miesiąc pod względem przychodu i dlaczego?
2. Które miasto jest najważniejszym rynkiem dla sprzedaży (liczba zamówień)?
3. Jaka jest optymalna godzina dnia na wyświetlanie reklam?
4. Jakie pary produktów są najczęściej kupowane razem?
5. Jaki produkt sprzedaje się najlepiej ilościowo i dlaczego?
